Данная программа сортирует строки на основе Radix Sort(как наиболее эффективная сортировка). Каждая строка отделена от следующей клавишей Enter.
Список слов вводится в левый столбец.
Для сортировки надо нажать кнопку Sort left area. 
Программа может сортировать и текстовые файлы. 
Для этого в панели меню нажмите кнопку Open  и выберете нужный файл, подождите загрузки файла и нажмите кнопку Sort left area. 
В случае ввода в строку файлов из русского и английского алфавита в конце строки будет оповещение об этом(появится (rus + eng)).
